# 武蔵攻略BASE — 受験対策ポータル

武蔵中学校 2027年2月入試に向けた学習アプリ群。すべてスタンドアロンHTML（オフライン対応）。

## 📦 アプリ一覧

| ファイル | 内容 | 問題数 |
|---------|------|--------|
| `index.html` | **ハブページ** — 全アプリへのナビゲーション | — |
| `musashi_note.html` | **武蔵ノート** v0.3 — 傾向分析・記述の型・漢字クイズ・2E対応 | 31カード / 45問 |
| `shakai.html` | **社会用語マスター** — 憲法・政治・歴史・現代社会 | 396問 |
| `kokugo.html` | **国語バトル** — 同音異義語・慣用句・四字熟語 | 77問 |
| `rika.html` | **理科バトル** — 生物・化学・物理・地学 | 69問 |

## 🚀 使い方

### GitHub Pages（推奨）
`https://yos-lab046.github.io/exam-app/` でアクセス

### ローカル
全HTMLをダウンロードしてダブルクリックで開く。Safari/Chrome対応。

## 📱 iPad対応
- ホーム画面に追加して「アプリ風」に使用可能
- localStorage で進捗を保持
- オフライン動作（フォント読込以外）

## 🛠 技術
- 全ファイルがスタンドアロンHTML（外部依存なし）
- CSS/JS インライン埋込
- localStorage で進捗管理
- Claude × Obsidian × NotebookLM で構築
